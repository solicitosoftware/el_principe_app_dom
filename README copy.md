## Install

yarn install

# or

npm install

## Getting Started

Para correr el proyecto:

yarn android

# or

yarn ios

Se despliega en un Equipo con sistema operativo Android, IOs o Maquinas Virtuales.

Para validar si existen dispositivos conectados

adb devices.

## Learn More.

app/App.js -- Stack.Navigator initialRouteName="login"
